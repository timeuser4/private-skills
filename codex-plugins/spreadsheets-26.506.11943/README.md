# Spreadsheets

Spreadsheets is the file-type wrapper plugin for spreadsheet artifact workflows.

This plugin is bundled into the Codex primary runtime.

## Included Skills

- `Spreadsheets`: create, edit, analyze, visualize, render, and export spreadsheet files such as `.xlsx`, `.xls`, `.csv`, and `.tsv`.

## Discoverability

Use this plugin for spreadsheet-oriented terms from the file-type naming model: sheet, sheets, Excel, CSV, model, spreadsheet, spreadsheets, workbook, tracker, and `.xlsx`.

## Source And Copy Flow

This plugin package is generated from `lib/artifacts/skills` and copied into `plugins/spreadsheets` through `lib/artifacts/skills/mapping.json`.
