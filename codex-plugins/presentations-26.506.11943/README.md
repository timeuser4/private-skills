# Presentations

Presentations is the file-type wrapper plugin for presentation slide deck workflows.

This plugin is bundled into the Codex primary runtime.

## Included Skills

- `Presentations`: create, edit, render, verify, and export editable presentation slide decks.

## Discoverability

Use this plugin for presentation-oriented terms from the file-type naming model: slides, deck, PowerPoint, presentation, presentations, PPT, and `.pptx`.

## Source And Copy Flow

This plugin package is generated from `lib/artifacts/skills` and copied into `plugins/presentations` through `lib/artifacts/skills/mapping.json`.
